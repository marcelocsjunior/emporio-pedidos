# Empório Pedidos

Sistema interno de gestão de pedidos, produção, entrega e fechamento mensal do Empório Restaurante.

> Repositório privado. Não versionar dados reais, bancos, tokens, secrets, credenciais, backups, anexos operacionais ou exportações de clientes.

O desenvolvimento ocorre por branches e pull requests. A branch `main` deve permanecer estável.
